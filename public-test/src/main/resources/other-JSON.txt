json：

{
     “rule":[
            {
                  "num":0,
                  "rate":1
            }
       ],
       "firstOrAvg":0,
}

字段说明：
rule:分享提成规则数组。若分享件数达到 `num`件，则获得 `rate` 的返点。其中，`num`的单位为`0.01`件，`num`取值为`200`时，表示`2`件。`rate`为返点，为整数。用户得到的返点为`件数 * 价格 * rate*0.01`

firstOrAvg:若用户购买某件商品，有多个分享成功记录，可以选择是返给在有效时间内最早分享成功的用户，或者平分给所有有效时间内分享成功的用户。`0`表示返给最早分享成功的用户，`1`表示平分

*参考自2-4组
